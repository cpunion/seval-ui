# seval-ui

S-expression based UI runtime for React.

## Packages

| Package | Description |
|---------|-------------|
| `@seval-ui/seval` | JavaScript-like language compiler and runtime |
| `@seval-ui/react` | React UI components |
| `@seval-ui/react-code` | Seval runtime for React |

## Installation

```bash
pnpm add @seval-ui/react @seval-ui/react-code
```

## Development

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm -r run build

# Run tests
pnpm -r run test

# Run demo
cd packages/demo && pnpm dev
```
